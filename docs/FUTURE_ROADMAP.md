# QueryCraft Future Roadmap & AI Implementation Ideas

이 문서는 QueryCraft의 확장성과 프로젝트 분석 역량 강화를 위한 장기 로드맵을 관리합니다.

## 1. 🔍 RCA (Root Cause Analysis) 시나리오 모드 (진행 중)
- **개념**: "매출 급감", "리텐션 하락" 등 가상의 비즈니스 장애 상황을 설정하고, 유저가 SQL을 통해 단계별로 원인을 파악하는 모드.
- **AI 역할**: 매일 다른 원인(특정 채널 효율 저하, 가입 페이지 버그 등)이 담긴 데이터와 시나리오 생성.
- **분석가 어필 포인트**: 가설 설정 및 데이터 기반 문제 해결 능력 증명.

## 2. 📝 AI 인사이트 리포트 자동화 (대기)
- **개념**: SQL 실행 결과 데이터를 AI에게 전달하여 "비즈니스 인사이트"와 "Action Plan"을 자동 생성.
- **AI 역할**: 쿼리 결과(숫자)를 브리핑하고 리포트 형태로 요약.
- **분석가 어필 포인트**: 데이터 추출을 넘어선 해석 자동화 역량.

## 3. 🎯 개인화 학습 추천 (Adaptive Learning) (대기)
- **개념**: 유저의 활동 기록(틀린 문제, 작성 시간, SQL 패턴)을 분석하여 맞춤형 학습 경로 제공.
- **AI 역할**: 유저의 약점을 파악하고 보완할 수 있는 다음 문제 추천.
- **분석가 어필 포인트**: 사용자 행동 데이터 기반의 의사결정 모형 라이브 적용.

## 4. 🗨️ Text-to-SQL 보조 도구 (대기)
- **개념**: 자연어 질문을 SQL 초안으로 변환해주는 에디터 기능.
- **AI 역할**: 비즈니스 질문의 맥락을 읽고 테이블 스키마에 맞는 쿼리 생성.
- **분석가 어필 포인트**: 데이터 민주화(Data Democratization)를 위한 기술적 해결책 제안.
