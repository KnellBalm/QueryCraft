# QueryCraft Documentation Wiki

QueryCraft 프로젝트의 지식 베이스입니다. 산재되어 있던 문서들을 주제별로 통폐합하여, 책처럼 읽을 수 있는 두 개의 핵심 Wiki와 에이전트 소통을 위한 관리 시스템으로 재구성했습니다.

---

## 🏛️ 1. Service Management (Active Communication)
*에이전트(AI)와 대화하며 실시간으로 프로젝트를 리빙하고 진행 상황을 관리하는 공간입니다.*

- [**FUTURE_PLAN.md**](./FUTURE_PLAN.md): 서비스의 장기적인 비전 및 AI 기능 로드맵.
- [**TODO_LIST.md**](./TODO_LIST.md): 현재 진행 중이거나 당장 수행해야 할 작업 목록.
- [**PATCH_NOTES.md**](./PATCH_NOTES.md): 완료된 작업들에 대한 기록 및 프로젝트 히스토리.
