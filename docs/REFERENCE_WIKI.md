## 1. 서비스 비전 및 개괄

QueryCraft는 단순한 SQL 학습 도구를 넘어, **AI 시대의 데이터 분석가가 일하는 방식**을 미리 경험하고 훈련하는 플랫폼입니다.

- **핵심 가치**: "실무형 데이터 탐색 능력 배양"
- **AI의 역할**: 문제 생성, 정답 피드백, 힌트 제공, 그리고 결과 해석까지 분석의 전 과정을 보조하고 리딩함.
- **주요 워크플로우**: Daily Challenge → RCA(원인 분석) → AI Lab(자동화 도구) → 스킬 분석 및 추천.

## 2. 2-Track 전략 (Expert vs Pioneer)

QueryCraft는 분석가의 역량을 두 가지 차원에서 정의하고 육성합니다.

| 구분 | **Track A: Core Skills (Expert)** | **Track B: Future Lab (Pioneer)** |
|:---:|:---|:---|
| **컨셉** | *"지금 당장 필요한 실무 역량의 완성"* | *"AI 시대 분석가가 일하는 방식의 미리보기"* |
| **목표** | SQL 숙련도, 데이터 문해력, 리포팅 기본기 | AI 협업, 의사결정 설계, 자동화 도구 활용 |
| **주요 기능** | Daily Challenge, Practice, Leaderboard | Text-to-SQL, AI Insight, RCA 시나리오, MCP |
| **사용자 경험** | Hard Training (반복, 숙달) | Advanced Simulation (실험, 탐구) |
| **UI 테마** | Blue/Grey (차분하고 전문적인 IDE) | Purple/Neon (혁신적이고 미래지향적인 Lab) |

## 3. 핵심 기능 상세 가이드

### 3.1 PA (Product Analytics) & Daily Challenge
매일 생성되는 가상의 비즈니스 데이터를 바탕으로 핵심 KPI를 도출하고 분석하는 훈련입니다.
- **목적**: KPI 도출, 퍼널 분석, 데이터 스키마 이해 능력 강화.
- **AI 활용**: 데이터 요약 기반 문제 생성 및 제출 결과에 대한 상세 피드백 제공.

### 3.2 RCA (Root Cause Analysis) 시나리오
이상 징후(매출 급감, 리텐션 하락 등)의 원인을 SQL로 규명하는 장애 분석 시뮬레이션입니다.
- **목적**: 비교 분석(기간/세그먼트/채널) 및 인과 관계 추론 역량 강화.
- **AI 활용**: 실제 장애 보고서 톤의 시나리오 구성 및 단계별 사고 가이드(힌트) 제공.

### 3.3 AI Lab (Text-to-SQL, Insight)
자연어로 데이터를 질문하고 결과를 비즈니스 언어로 해석받는 지능형 분석 도구입니다.
- **목적**: 분석 작업의 자동화(Automation) 및 생산성 극대화 경험.
- **AI 활용**: 스키마 기반 SQL 자동 생성(Text-to-SQL) 및 액션 아이템 제안(AI Insight).

### 3.4 Adaptive Learning (맞춤형 추천)
사용자의 활동 기록과 오답 패턴을 분석하여 최적의 학습 경로를 설계합니다.
- **목적**: 학습 효율 최적화 및 자기 객관화(약점 파악).
- **AI 활용**: JOIN, Window Function 등 약점 영역 자동 탐지 및 맞춤형 문제 추천.

## 4. 데이터 및 실무 활용 시나리오

### 4.1 실무 적용 사례
- **신입 분석가 온보딩**: Text-to-SQL을 보조 도구로 사용하여 쿼리 작성 시간을 단축하고 문법을 학습합니다.
- **보고서 자동화**: AI Insight 기능을 활용하여 쿼리 결과에서 핵심 발견(Key Findings)을 추출하고 리포트 초안을 작성합니다.
- **포트폴리오 강화**: RCA 모드에서 작성된 분석 리포트를 바탕으로 장애 대응 역량을 증명합니다.

### 4.2 분석 업무와의 연결
- **Mixpanel 연동**: 사용자의 모든 학습 활동은 Mixpanel을 통해 트래킹되며, 이를 통해 학습 지속성과 성과를 정량적으로 관리합니다.
- **실험 데이터**: 프롬프트 버전 및 실험 그룹별 성과 분석을 통해 AI 생성 콘텐츠의 품질을 직접 운영/검증해 볼 수 있습니다.
